# SiraMangagerAPI
Sira is an Amharic word meaning "Work". SiraMangager is sira managing application that helps you in doing your sira in well organized and managed manner.

## Installation 💻

Install SiraManager-frontend with npm

```bash
  cd SiraManagerAPI
  npm install 
  npm start
```
